# Linux Storage Investigation

---

## Check Disks

```bash
lsblk

lsblk -f

blkid

fdisk -l
```

---

## Check PCI Devices

```bash
lspci | grep -i "Non-Volatile"
```

---

## Check Kernel

```bash
dmesg | grep -Ei "nvme|pci|error|timeout"
```

---

## Hardware Validation

Compare with a healthy worker.

Check

- Missing NVMe device
- Missing PCI controller
- Hardware errors
